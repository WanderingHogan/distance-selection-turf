Process

1. We use D3.js to read a CSV file and display the data as GeoJSON on the map.

2. The user can click a point, and that point is then turned in to a turf.point object. (show)

3. We then create two buffers of the point, one for the inner number and outer buffer you specified. (show)

4. We use turf.erase to remove the inner polygon from the outer polygon, creating a donut! (show)

5. Now we use turf.within to extract just the GeoJSON points from the donut. (show)

6. Finally, we use turf.nearest to compare the original point and the points that intersected the donut, returning the nearest point within the inner and outer ranges.

Built with: turfjs, leafletjs, bootstrap, jquery, d3js. Paper theme from Bootswatch.
